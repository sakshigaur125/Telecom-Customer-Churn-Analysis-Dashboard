## Overview
This project analyzes telecom customer churn using an interactive Excel dashboard. It highlights the main drivers of churn and helps identify customer groups that are most at risk of leaving.

## Business Problem
The key problem is customer attrition, especially among short-term contract users, electronic check users, and customers without tech support. This creates avoidable revenue loss and weakens customer retention.

## Solution
The dashboard converts raw customer data into clear business insights. It helps the business focus on the right retention actions, including encouraging longer contracts, improving support, and targeting risky customers early.

## Key Insights
- Month-to-month customers show the highest churn risk.
- Electronic check users are significantly more likely to leave.
- Customers without tech support are more likely to churn.
- Early-tenure customers need closer retention attention.

## Dashboard Features
- KPI cards for Total Customers and Churn Rate
- Churn analysis by Contract
- Churn analysis by Tech Support
- Churn analysis by Payment Method
- Churn analysis by Tenure Groups
- Interactive slicers for filtering the dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dataset
Telecom customer churn data including customer details, tenure, contract type, payment method, monthly charges, total charges, support usage, and churn status.
